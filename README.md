# Летняя парктика на 1 курсе. 

## Описание
Web-платформа по тестированию студентов.

  ### Фронт - html/css/vuejs

  ### Бэк - nginx + nodejs

## Для того, чтобы запустить:

#### Установить Node.js
  ```
  sudo apt-get install nodejs
  ```
#### Запустить сервер
  ```
  node index.js
  ```
#### Зайти на localhost:8080
